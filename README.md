# 交医-华西Journal Club

## 项目简介

这是交医-华西 Journal Club 的网站项目，用于展示研讨会信息、讨论议题、日程安排以及活动记录。网站主要关注基因组学和医学研究领域的学术交流。

## 网站结构

网站包含以下几个主要部分：

1. **封面区域** - 展示 Journal Club 的标题和简介
2. **讨论议题** - 介绍当前的研究主题，如全基因组关联分析(GWAS)
3. **日程安排** - 显示未来的会议安排时间表
4. **活动记录** - 记录过去的会议内容，包括日期、讲者和文章名称

## 技术栈

- HTML5
- CSS3
- Font Awesome 5.15.4 (用于图标)

## 文件结构

- index.html - 网站主页面
- styles.css - 样式表文件

## 更新维护

网站内容定期更新，包括：
- 添加新的议题讨论
- 更新日程安排
- 记录新的活动信息
